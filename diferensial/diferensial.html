Solusi Numerik dari Persamaan Diferensial Biasa¶

pengertian persamaan diferensial biasa¶

Persamaan diferensial biasa adalah persamaan diferensial yang fungsinya tidak diketahui (variabel terikat) yang merupakan fungsi dari satu variabel bebas. Dalam bentuk sederhananya, fungsi yang tidak diketahui adalah fungsi real ataupun fungsi kompleks, namun biasanya berupa fungsi vektor atau matriks. tidak hanya itu, persamaan diferensial biasa diklasifikasikan menurut turunan tertinggi dari variabel dependen yang muncul dalam persamaan.[1]

contoh persamaan diferensial biasa. Metode numerik untuk persamaan diferensial biasa merupakan metode yang digunakan untuk mencari pendekatan numerik dari solusi persamaan diferensial biasa (ODE). Penggunaannya juga disebut "integrasi numerik", walaupun istilah ini juga dapat merujuk pada perhitungan integral.

Banyak persamaan diferensial tidak bisa diselesaikan menggunakan perhitungan simbolis (analisis). Namun, untuk tujuan praktis seperti hal-nya dalam bidang teknik menggunakan metode numerik penyelesaian biasanya cukup. Algoritma yang dipelajari di sini dapat digunakan untuk menghitung perkiraan ini. Metode lain adalah dengan menggunakan teknik kalkulus untuk menemukan solusi dari ekspansi deret.

Persamaan diferensial umum di banyak disiplin ilmu, termasuk fisika, kimia, biologi, dan ekonomi. Selain itu, beberapa metode dalam persamaan diferensial parsial numerik mengubah persamaan diferensial parsial menjadi persamaan diferensial biasa, yang kemudian harus diselesaikan.

Berikut ini merupakan contoh sederhana dari hukum gerak kedua Newton, yang mana menghasilkan persamaan diferensial.[2]

metode Euler untuk persamaan diferensial orde 1¶

Dapat dinyatakan secara umum dalam dua bentuk, yaitu:

  1. Bentuk Implisit

  1. Bentuk Eksplisit Seperti halnya dalam matakuliah Kalkulus: y' merupakan turunan pertama dari y terhadap variabel x. y'' merupakan turunan kedua dari y terhadap variabel x. Dst

Jikalau diberikan persamaan diferensial biasa orde satu 

    y' = dy/dx = f(x, y); y(x0) = y

Misal :

    yr = y(xr)

merupakan hampiran nilai y di xr yang dihitung menggunakan metode Euler. maka,

    xr = x0 + rh, r = 0,1,2,... n.

Algoritma persamaan diferensial biasa¶

Algoritma metode Euler untuk persamaan diferensial orde 1 sebagai berikut:

- input y0, x0 = nilai awal, titik awal
- xn=b = batas interval
- N = jumlah subinterval

Definisi  fungsi f(x,y(x)) = untuk sisi kanan persamaan diferensial

x:=x0, y:=y0

h:=(xn-y0) / n

for n: = 1 to n

        {y:=y+h f(x,y(x))

        x:= x0+nh

        hasil: x,y}
Penerapan code program menggunakan python¶

Berikut ini merupakan code program menggunakan metode Euler untuk persamaan diferensial orde 1 

import numpy as np
import matplotlib.pyplot as plt

def func(N,t):
    k= 0.02
    return -k*N

N0 = 1000
t0 = 0

ndata = 10
tn = 100
t = np.linspace(t0,tn,ndata)
N = np.zeros(ndata)
N[0] = N0
h = t[1]-t[0]

for i in range(0,ndata-1):
    N[i+1] = N[i]+h*func(N[i],t[i])

k =0.02    
NA = N0*np.exp(-k*t)

plt.plot(t,NA,'red',label='analitik')
plt.plot(t,N,'x',color= 'black', label='Euler')
plt.show()
