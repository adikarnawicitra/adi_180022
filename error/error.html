Kesalahan dalam perhitungan numerik¶

Error(galat) dalam metode numerik merupakan selisih yang diperoleh antara nilai sebenarnya dengan nilai yang dihasilkan dari metode numerik. Dalam metode numerik, hasil yang didapatkan bukanlah hasil yang sama persis dengan nilai sebenarnya. Akan selalu ada selisih, dikarenakan hasil yang diperoleh dengan metode numerik adalah hasil yang didapat dengan proses iterasi (loop) agar mendekati nilai sebenarnya. meskipun demikian bukan berarti hasil yang diperoleh dari metode numerik tersebut salah, karena error tersebut dapat ditekan sekecil mungkin hingga hasil yang didapat sangat mendekati nilai sebenarnya atau bisa dikatakan kesalahan mendekati nol.

Metode numerik adalah teknik pemecahan masalah yang menggunakan operasi aritmatika (yaitu operasi penjumlahan, pengurangan, perkalian, dan pembagian) untuk mengungkapkan masalah secara matematis. Metode ini digunakan karena banyak masalah matematika yang tidak dapat diselesaikan dengan menggunakan metode analitik. Bahkan dengan solusi analitik, prosesnya pun seringkali cukup rumit dan memakan waktu, sehingga efisiensinya rendah.

Perhitungan dalam metode numerik merupakan perhitungan dengan mengulang perhitungan dengan terus-menerus untuk mendapatkan hasil yang mendekati nilai penyelesaian yang akurat(exact). Dengan cara ini tentunya setiap nilai yang dihitung akan memiliki nilai error. Dalam analisis metode numerik, kesalahan ini menjadi sangat penting. Karena kesalahan penggunaan algoritma pendekatan akan mengakibatkan nilai kesalahan yang besar, hal ini tentunya tidak diharapkan. Oleh karena itu, metode analisis selalu membahas tingkat kesalahan dan tingkat kecepatan pemrosesan yang akan terjadi.[1]

Error Numerik¶

Kesalahan numerik merupakan error yang disebabkan oleh proses pendekatan atau hampiran. Berikut merupakan faktor-faktor untuk kesalahan metode numerik, termasuk :

  • Kesalahan bawaan (*inherent error*) merupakan kesalahan yang diakibatkan oleh pengukuran, kesalahan manusia (seperti kesalahan pencatatan) atau tidak memahami hukum fisika dari data pengukuran.

  • Kesalahan pembulatan (*rounding*) merupakan kesalahan yang disebabkan karena pembulatan. Contoh: 3.142857143... menjadi 3.14.

  • Kesalahan pemotongan (*Chopping*) merupakan kesalahan yang diakibatkan ketika jumlah angka efektif dikurangi. Contoh : 0.6666666... menjadi 0.66.[2]

  1. yaitu Angka 1 sampai 9.

  2. Abaikanlah angka 0 setelah koma sebelum angka 1 sampai 9.

Misal : 0,0005813 memiliki 4 (empat) angka penting, dan 0,700124 memiliki 6 (enam) angka penting.

Metode pengukuran akurasi

Kesalahan mutlak atau absolut suatu angka, pengukuran ataupun perhitungan adalah pembeda antara numerik nilai sebenarnya terhadap nilai pendekatan yang diberikan, atau nilai yang diperoleh dari perhitungan atau hasil pengukuran.

Rumusnya :

Kesalahan = Nilai Eksak - Nilai perkiraan ket :

E = Error absolut

P = Nilai Eksak

P* = Nilai perkiraan

Kesalahan relatif merupakan kesalahan mutlak yang dibandingkan dengan nilai eksak yang terjadi. Gunakan rumus berikut untuk menghitung kesalahan relatif:

e = Error relatif terhadap eksak

E = Error absolut

P = Nilai Eksak

P*= Nilai perkiraan[3]

Algoritma menggunakan metode bisection¶

Metode bisection merupakan metode penyelesaian integral numerik sederhana untuk pencarian akar persamaan (metode bagi dua) . Berikut ini merupakan langkah-langkah algoritma bisection :

``` 1.Tentukan fungsi (x) dan nilai error

2.Tentukan titik a dan b pada fungsi tersebut

3.Tentukan titik tengah (c) dengan membagi dua antara penjumlahan titik a dan b

         c=a+b/2

4.Masukkan titik a, b dan c pada fungsi x

        f(a),f(b),f(c)

5.kalikan f(a) dan f(c),kemudian jadikan nilai mutlak

        |f(a,c)|=f(a)xf(c)

6.jika nilai mutlak f(a,c) < nilai error maka perulangan berhenti

7.akan tetap jika > nilai error maka tentukan lagi apakah nilai f(a,c) yang diperoleh > atau < dari 0

8.jika nilai f(a,c) < 0 maka titik b menjadi titik c, jika nilai f(a,c) > 0 maka titik a menjadi titik c

9.Ulangi tahap tersebut sampai nilai mutlak dari f(a,c) < nilai error ```

Penerapan code program menggunakan python¶

Berikut ini merupakan code program menggunakan metode bisection

def f(x):
    y = x**2-4
    return y

def bisection(f,a,b):
    toleransi = 0.001
    if f(a)*f(b)>0:
        print ('a dan b tidak mengandung akar')
        return None
    c = (a+b)/2
    while abs(f(c))>toleransi:
        c = (a+b)/2
        if f(a)*f(c)>0:
            a = c
        else:
            b = c
        print('akar yang mendekati: ',c)
    return c

a = float(input('masukkan nilai a: '))
b = float(input('masukkan nilai b: '))

x = bisection(f,a,b)
print()
print('akar persamaan x = ',format('%.2f'%x))

mencari pendekatan fungsi e^x-cos(x) untuk x=1 dan batas = 0.001

import math 
from math import factorial, exp, cos

def exponensial (x, n):
    hasil = 0.0
    for i in range (0, n):
        hasil = hasil + x**i/factorial(i)
    return hasil

oldval = 0.0
x = 1
n = 1
trueval = exp (x)-cos(x)
print ("nilai asli :","%.8f"%trueval)
print('faktor','\t','estimasi','\t','true error','\t','error relatif')
ca =1
while(ca>0.001):
    val = exponensial (x, n)-cos(x)
    ct = abs((val-trueval)/trueval)*100
    ca = abs((val-oldval)/val)*100
    oldval = val

    print (n ,'\t',"%.8f"%val,'\t',"%.8f"%ct,'\t',"%.8f"%ca)
    n+=1
print ("nilai yang mendekati :", "%.8f"%val,"dengan error","%.8f"%ct)
